# 交互界面设计规范

---

## 第一章	设计理念

>- 主题：For Simple

- 避免使用各类SVG图标。

  >1. 寻找或绘制风格、尺寸统一的SVG图标会耗费大量的时间和精力，由于是我个人开发因此我应将有限的时间转移到更重要的工作上。
  >2. 各类SVG图标资源会增加软件的硬件占用资源。
  
- 避免过度的使用动画：

  >1. 过度使用动画会消耗大量资源，造成程序卡顿。
  >2. 目前动画的使用场景包括：进场动画、退场动画和少量的循环动画。

## 第二章	尺寸设计

#### 第一节	尺寸单位

- 像素【px】：屏幕上最基本的显示单位，一个像素就是屏幕上最小的可显示点。不同的屏幕其像素点大小和个数不同。

#### 第二节	尺寸计算

- 屏幕宽度：在QML中可以通过下面的代码获取屏幕的宽度信息，其单位为px

  ```js
  readonly property real screenWidth: Screen.width
  ```

- 屏幕高度：在QML中可以通过下面的代码获取屏幕的高度信息，其单位为px

  ```js
  readonly property real screenHeight: Screen.height
  ```

- 屏幕尺寸：以屏幕斜对角线长度来衡量屏幕的大小，通常情况下其单位为inch，在QML中通过屏幕宽度和屏幕高度计算获得所以其单位为px

  ```
  readonly property real screenSize: Math.sqrt(screenWidth*screenWidth+screenHeight*screenHeight)
  ```

- 窗口缩放比例：桌面端应用程序可以全屏显示也可以不全屏显示，其通过窗口缩放比例进行控制，其取值范围为：0~1.0

  ```js
  property real windowScale: 0.8
  ```

- 窗口宽度：应用程序实际宽度，单位：px

  ```js
  property real windowWidth: screenWidth * windowScale
  ```

- 窗口高度：应用程序实际高度，单位：px

  ```
  property real windowHeight: screenHeight * windowScale
  ```

- 窗口尺寸：以窗口斜对角线长度来横向应用程序界面尺寸大小，单位：px

  ```
  property real windowSize: Math.sqrt(windowWidth * windowWidth + windowHeight * windowHeight)
  ```

- 窗口圆角半径：其相关控件圆角尺寸半径，单位：px

  ```
  property real windowRadius: windowSize * 0.01
  ```

## 第三章	颜色搭配

#### 第一节	颜色体系

- 主色【Primary】：

  >1. UI组件核心元素的颜色配置
  >2. 数量控制在一到两种

- 辅助色【Secondary】：

  >1. UI组件相关元素的强调、高亮和警示

- 背景色【Background】：

  >1. UI界面或组件的背景颜色
  >2. 其相关颜色具有亮度层次关系

- 文字色【Text】：

  >1. 用于配置主文字、次文字、禁用文字的颜色

- 状态色【State】：

  >1. 表征UI组件相关状态：成功（绿色）、警告（黄色）、错误（红色）、信息（蓝色）

#### 第二节	颜色选择

```js
    //1、常用颜色
    /// transparent 透明
    readonly property color transparent: "transparent"
    //2、主色
    /// primary1 用于UI组件的主要配色
    readonly property color primary1: "#41F0F0"
    /// primary2 用于UI组件的状态变化后配色，如：按下、激活等
    readonly property color primary2: "#00B0B0"
    /// primary3 用于发光
    readonly property color primary3: "#7CFFFF"
    /// primary4 用于发光：白
    readonly property color primary4: "#FFFFFF"
    //3、辅助色
    /// secondary1 用于提示
    readonly property color secondary1: "#FF9500"
    /// secondary2 比辅助色1更亮
    readonly property color secondary2: "#FFC870"
    //4、背景色
    /// secondary1 背景最底层颜色
    readonly property color background1: "#010308"
    /// secondary2 整体主背景
    readonly property color background2: "#020609"
    /// secondary3 核心面板背景
    readonly property color background3: "#050C11"
    /// secondary4 比主背景稍亮，用于区分面板层级
    readonly property color background4: "#0A1620"
    /// secondary5 给HUD浮层、按钮背景使用，增强前景感
    readonly property color background5: "#13232F"
    /// secondary6 HUD中发光元素的衬底，让光更突出
    readonly property color background6: "#1A2B38"
    //5、文字颜色
    /// font1 主要文本色
    readonly property color font1: "#E0F7FA"
    /// font2 次要文本色
    readonly property color font2: "#A0C0C0"
    /// font3 禁止文本色
    readonly property color font3: "#506070"
    //6、状态色
    /// success 标志状态正常
    readonly property color success: "#00FF88"
    /// warning 表征警告
    readonly property color warning: "#FFAA00"
    /// error 标志状态错误
    readonly property color error: "#FF3300"
    /// normal 标志状态正常
    readonly property color normal: "#41F0F0"
```
